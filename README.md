
Tic-Tac-Toe
------------

A classic game able to be played in standard mode or 25-square mode, and both can be played in Pacman theme.


Link to live site: https://dutchessoflx.github.io/tic-tac-toe/



Technology used:
* Html
* Css
* Javascript
* JQuery
